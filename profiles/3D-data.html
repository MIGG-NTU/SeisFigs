Read netCDF¶

读 netCDF 文件:

  • 查询头段信息: ncdump -c 网格文件 | ncdump -h 网格文件

  • convert: 表数据格式转换、列提取、列粘贴

    • 三维网格文件 中提取一维变量数据

      • gmt convert "FWEA18_kmps.nc?depth" &> depth

  • grdinfo: 提取二维网格文件的基本信息

  • grdconvert: 网格文件转换为其它网格文件格式

    • 三维网格文件 中提取第一层数据(二维网格数据)

      • gmt grdconvert "FWEA18_kmps.nc?vph[0]" vph0.nc -V

Data Format Tranformation¶

  • grd2xyz: 将网格文件转换成表数据

  • xyz2grd: 将表数据转换成网格文件，不具备网格化/插值功能

Data Gridding¶

  • surface: 网格化表数据

    • gmt surface tabledata -Rlon1/lon2/lat2/lat2 -I1/1 -Ggridfile

Data Profile Cutting¶

  • project: 生成测线、将数据点投影到测线上

  • grdtrack: 沿着指定的位置采样二维网格文件，指定的位置可以是由 project 生成的测线

    • 注意提取前可以用 surface 对网格数据进行插值
